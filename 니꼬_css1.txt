1. html과 css 파일 연결하는법

  1-1 첫번째 방법 <head> 안에 <link href="파일명이나 주소" rel="stylesheet" /> 
  rel은 관계를 말한다 <-- 추천

  1-2 두번째 방법은 html 파일속 head에 <style></style>를 만들고 그안에 작성하는방법

  2. css 기본 작성법

  예 : h1 {                  예 :  태그 {
          color: blue;                    속성 : 값;
          font-size: 20px;         ]
        }

3. css 적용 메커니즘

  css는 위에서부터 아래로 적용이 되어서 마지막에 적용이된값이 최종값이된다.

4. html에서 공간의형태 두종류 block 와 inline

  block : 옆에 아무것도 오지못하는 공간 예: div, main, header 등등 개많음
  inline : 옆에 다른 태그가 들어올수 있다. 예: span, link, img

5. 공간의 형태를 바꾸기 block -> inline, inline -> block
  예:
  div {                  
          display: inline;
          color: blue;                  
          heigth: 150px         
        }

  display 속성으로 공간 속성을 바꿀수가 있다. block -> inline, inline -> block
  하지만 block는 높이와 너비가 있지만 inline 으로 바꾸게 되면 높이와 너비가 사라진다
  반대로 inline 은 높이와 너비가 없지만 block로 바뀌면 높이와 너비가 생긴다.

6. block의 특징 3가지 margin, padding, border

6-1 margin

  margin : margin은 box의 border(경계)로부터 바깥에 있는 공간이다
  종류 : margin, margin-top, margin-bottom, margin-left, margin-right
  예 : margin: 20px 15px;  // 해석 : margin: 상하20px 좌우15px;  
  예2 : margin: 12px 3px 6px 9px; 해석 : margin: 상 우 하 좌 // 시계방향순  
  Collapsing margin 현상 (상하에서만 발생함)
  div margin이 body margin과 같을 때 발생하고 body 와 div margin은 하나로 취급된다.    

6-2 padding

  padding : margin과 반대로 border(경계)로부터 안쪽에 있는 다른 블록까지의 공간

6-3 border

  border : border은 경계 그자체를 말한다 직사각형이 있으면 직사각형의 4가지 변 그자체의 스타일이다
  예 : border: 1px dashed black; // 해설 : border: 선두께 선디자인 선의색깔
  전체요소에 스타일을 추가하는방법 : css에 * {    }

7. id와 class의 특성

7-1 id

  id : id는 고유의 특성이다 . 예 : <div id="hi"></div>    <- div의 id가 hi 라는 것이다.
  id불러오기 : css에서 특정 id만 불러오고 싶을때는 예 : #hi {  속성: 값; } 요런식으로

7-2 class

  class : class는 여러개의 태그들이 공용으로 사용할수있는 포인터다. 
  예 : .class{  속성: 값; } 
  각 클래스와 id는 여러개를 동시에 적용시킬수 있다 
  예 : .class.class2.class3 { }  <-  여러 클래스를 동시에 적용가능
  예2 : #id#id3#id4 {  }   <- 여러 id 적용 가능

8. inline의 특징

  inline : inline은 padding을 쓸경우 상하좌우 다 적용이 되지만 margin을 적용 시킬경우
  inline의 특성 높이와 너비가 없듯이 좌우만 margin이 적용된다.
